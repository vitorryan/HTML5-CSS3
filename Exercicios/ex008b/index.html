Código-fonte

O comando document.getElementById('teste') é escrito em JavaScrip

O comando a cima não é uma fonte mono espaçada.
Agora o comando com a fonte mono espaçada usando a tag code, fica assim:
document.getElementById('teste')

A fonte mono espaçada é ótima para ler códigos, porém ele não organiza o codigo com indentação.

Sem indentação:
num = int(input('Digite um número')) if num % 2 == 0: print(f'0 número {num} é PAR) else: print(f'0 número {num} é ÍMPAR) print('Fim do programa')

Com indentação: 

                    
                        num = int(input('Digite um número'))
                        if num % 2 == 0:
                            print(f'0 número {num} é PAR)
                        else:
                            print(f'0 número {num} é ÍMPAR)
                        print('Fim do programa')

Com indentação, sem o espaço: 

                
num = int(input('Digite um número'))
if num % 2 == 0:
    print(f'0 número {num} é PAR)
else:
    print(f'0 número {num} é ÍMPAR)
print('Fim do programa')
