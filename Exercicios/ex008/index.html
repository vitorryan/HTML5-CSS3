Negrito / destaque

Existem duas maneiras para se colocar o negrito, uma meneira Semantica e outra não semantica.
Nesta frase teremos uma palavra em negrito. Usando a Tag B (não Semántica)

Nesta frase teremos uma palavra em destaque. Usando a Tag STRONG (Semántica)

Itálico / Énfase

Nesta frase teremos uma palavra em Italico, usando a Tag I (Não semántica)

Nesta frase teremos uma palavra em Énfase, unsando a Tag EM (Não semántica)

Quando usar Negrito e Itálico

Negrito e Itálico, são mais para forma, que é mais usando em CSS3

Já no HTML5 usamos o Destaque e Énfase para alguma palavra ou frase 'forte'

Usando Destaque e Énsafe na mesma frase

No curso o professor não comentou nada sobre isso nesta aula. Porém fazendo alguns testes descobri que se pode usar o Destaque e Énfase juntos, colocando uma Tag dentro da outra Tag
